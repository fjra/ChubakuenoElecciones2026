## Sobre los datos

Los datos de actas OCR incluidos en este repositorio se proveen **tal cual (as-is)**, sin garantía de ningún tipo, ya sea explícita o implícita. Aproximadamente 1400 de las actas escaneadas han sido verificadas manualmente, el resto proviene directamente del procesamiento OCR sin revisión humana.

Cualquier sugerencia, corrección o contribución es bienvenida — abre un issue o un pull request.
